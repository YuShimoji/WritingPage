# Google Keep 双方向連携 仕様書 (SP-075)

## 概要

Google Keep との双方向データ連携を提供する。
Keep のラベルごとにメモを一括取り込み、
Zen Writer の断片・メモをKeepに送出できるようにする。

執筆の素材管理ツールとしてKeepを活用し、
アイデアの収集と作品への統合をシームレスにする。

---

## 背景課題

- 執筆のアイデアメモは Google Keep に散在しているが、手動でコピペしている
- Keep のラベルで分類したメモ群を、一括で作品のセクションに取り込む手段がない
- Zen Writer で書いた断片メモを Keep に戻して、モバイルで確認・追記する手段がない
- Keep と Zen Writer の間でデータを行き来させる公式な手段がない

---

## 目的

- Google Keep のメモをラベル単位で Zen Writer に取り込めるようにする
- Zen Writer の章・断片を Google Keep に送出できるようにする
- 取り込み / 送出の操作を簡単にする（ワンクリック同期ではなく明示的操作）

---

## 技術的前提

### Google Keep API の現状

- Google Keep には公式の公開 REST API が存在しない（2025年時点）
- Google Keep API は Workspace Enterprise 向けに限定提供
- 代替手段:
  1. **Google Takeout**: Keep データをJSON形式でエクスポート（手動）
  2. **Google Keep → Google Docs 連携**: Keep のメモを Docs に送信し、Docs API 経由で取得
  3. **非公式API / スクレイピング**: 安定性・利用規約の問題あり
  4. **間接連携**: Keep のメモをコピーしてZen Writerにペースト（クリップボード経由）

## 段階的実装

### Phase 1: インポート (Takeout ベース)

1. ユーザーが Google Takeout で Keep データをダウンロード (JSON)
2. Zen Writer にインポート機能を追加
3. ラベルでフィルタリングし、選択したメモを取り込む

#### インポートUI

```
┌─ Keep メモ インポート ─────────────────┐
│                                      │
│  ファイル: [Keep-data.zip を選択]       │
│                                      │
│  ラベル:                              │
│  ☑ 小説アイデア (12件)                │
│  ☐ 買い物 (3件)                      │
│  ☑ キャラ設定 (8件)                   │
│  ☐ アーカイブ済み (25件)              │
│                                      │
│  取り込み先: [▼ 新しい章として追加]     │
│  - 新しい章として追加                  │
│  - 選択中の章に挿入                   │
│  - 素材メモとして取り込み              │
│                                      │
│  [キャンセル]          [インポート]     │
└──────────────────────────────────────┘
```

#### Keep JSONフォーマット

```json
{
  "color": "DEFAULT",
  "isTrashed": false,
  "isPinned": false,
  "isArchived": false,
  "textContent": "メモの本文",
  "title": "メモのタイトル",
  "userEditedTimestampUsec": 1234567890000000,
  "labels": [
    { "name": "小説アイデア" }
  ]
}
```

### Phase 2: エクスポート (テキスト / Markdown)

Zen Writer の章やメモを Keep に取り込める形式でエクスポート:

1. 章を選択して「Keep向けエクスポート」
2. 1章 = 1メモとしてテキストファイルを生成
3. ユーザーが手動で Keep にコピーまたはGoogle Docs経由で追加

#### エクスポートUI

- チャプターリストの右クリックメニューに「Keep用にコピー」を追加
- プレーンテキスト形式でクリップボードにコピー
- 複数章選択時は区切り付きで一括コピー

### Phase 3: リアルタイム連携 (将来)

Google Keep APIが公開された場合、または Google Tasks API / Google Docs API 経由での間接連携:

- OAuth2 認証
- ラベルベースの双方向同期
- 変更検知と差分同期
- 競合解決UI

## データマッピング

| Keep | Zen Writer | 方向 |
|------|-----------|------|
| メモ (note) | 章 / セクション / 素材メモ | Keep → ZW |
| タイトル | 章タイトル / 見出し | 双方向 |
| 本文 | 章の本文 (Markdown変換) | 双方向 |
| ラベル | タグ / カテゴリ | Keep → ZW |
| チェックリスト | タスクリスト (Markdown) | Keep → ZW |
| 画像 | インライン画像 | Keep → ZW |
| 色 | メタ情報として保存（表示には使わない） | Keep → ZW |

---

## 成功状態

### Phase 1

- Google Takeout の Keep データ (ZIP/JSON) をインポートできる
- ラベルでフィルタリングして取り込み対象を選択できる
- 取り込んだメモが章または素材メモとして追加される

### Phase 2

- 章をプレーンテキスト形式でクリップボードにコピーできる
- 複数章を一括でエクスポートできる
