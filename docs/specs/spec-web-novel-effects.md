# Web小説演出統合 仕様書 (SP-074)

## 概要

Web小説 / ビジュアルノベル / アドベンチャーノベルに必要な演出機能を統合する。
テクスチャオーバーレイ、文字アニメーション、タイピング演出、SE（効果音）、
顔アイコン付きダイアログをカバーし、執筆時（WYSIWYG）からリアルタイムにプレビューできる。

SP-062 (テキスト表現アーキテクチャ) を基盤とし、その責務モデル上に構築する。
将来的にプリセットとして機能を取り出しやすい構造を維持する。

---

## 背景課題

- テキストアニメーション (SP-062) は文字単位の装飾・アニメーションクラスの投影まで定義しているが、
  テクスチャ重畳や音声連動は未定義
- アドベンチャーノベル的な顔アイコン + 吹き出しの構造がない
- タイピング演出（1文字ずつ表示）は読者向け出力の仕組みとして未定義
- SE（効果音）のトリガーと管理の仕組みがない
- これらが個別に散在すると、組み合わせたときの競合が管理できない

---

## 目的

- 執筆時WYSIWYGで演出効果をリアルタイムプレビューできるようにする
- 読者向け出力（HTML）で同じ演出が再生されるようにする
- SP-062 の責務モデル (TextEffect / Animation / Ornament / Preset) に統合する
- 将来のプリセット化を見据えた拡張可能な構造にする

---

## SP-062 との関係

SP-062 の責務モデルに以下を追加拡張する:

| 責務 | SP-062 (既存) | SP-074 (追加) |
|------|-------------|--------------|
| TextEffect | 文字単位装飾class | テクスチャオーバーレイ |
| Animation | 文字/box単位アニメclass | タイピング演出、スクロールトリガー |
| Ornament | box静的見た目class | ダイアログボックス（顔アイコン付き） |
| Preset | sugar展開 | ジャンルプリセット（ADV風、Web小説風等） |
| **新規: Media** | — | SE（効果音）のトリガーと再生管理 |

## 機能仕様

### 1. テクスチャオーバーレイ

文字の上にテクスチャ（画像/パターン/グラデーション）を重ねてアニメーションさせる。

#### 1.1 対応テクスチャ

| タイプ | 説明 | WYSIWYG対応 |
|--------|------|-------------|
| wave | 波のようなテクスチャアニメーション | CSS animation |
| sparkle | キラキラのパーティクル | CSS animation + pseudo-elements |
| cosmic | 宇宙的なグラデーション流動 | CSS gradient animation |
| fire | 炎のゆらめき | CSS animation |
| glitch | デジタルグリッチ | CSS animation |
| custom | ユーザー定義画像パターン | CSS mask + background |

#### 1.2 記法

既存の `[tag]...[/tag]` パターンに統一:

```markdown
[wave]波打つテキスト[/wave]
[sparkle]キラキラ[/sparkle]
[cosmic]宇宙的テキスト[/cosmic]
[fire]炎のテキスト[/fire]
[glitch]グリッチテキスト[/glitch]
```

#### 1.3 CSS実装方針

- `background-clip: text` + `background-image` でテクスチャを文字に適用
- アニメーションは `@keyframes` + CSS custom properties で制御
- reduced motion 時は静的テクスチャ表示に縮退（SP-062 の縮退規則に従う）

### 2. タイピング演出

テキストが1文字ずつ表示される演出。

#### 2.1 モード

| モード | 動作 |
|--------|------|
| auto | 指定速度で自動再生 |
| click | クリック/タップで次の文/段落を表示 |
| scroll | スクロール位置に連動して表示 |

#### 2.2 記法

```markdown
:::zw-typing{speed:"50ms", mode:"auto"}
一文字ずつ表示されるテキスト。
:::
```

#### 2.3 WYSIWYG上の表示

- 執筆時: テキストは通常表示。ボックスの角にタイピング演出アイコンを表示
- プレビュー時: 実際のタイピングアニメーションが再生
- 再生コントロール: プレビュー内に再生/停止/リセットボタン

### 3. ダイアログボックス（顔アイコン付き）

アドベンチャーノベル風の発言表示。

#### 3.1 構造

```markdown
:::zw-dialog{speaker:"アリス", icon:"alice.png", position:"left"}
こんにちは。ここが不思議の国ね。
:::
```

#### 3.2 表示

```
┌────────────────────────────────────┐
│ ┌─────┐                           │
│ │ icon │  アリス                    │
│ │     │  こんにちは。ここが不思議の国ね。│
│ └─────┘                           │
└────────────────────────────────────┘
```

| プロパティ | 説明 |
|-----------|------|
| speaker | 話者名 |
| icon | 顔アイコン画像パス（省略可） |
| position | left / right / center |
| style | default / bubble / bordered / transparent |

#### 3.3 WYSIWYG上の表示

- 執筆時: 実際のダイアログボックスとして表示（ブロック要素）
- アイコンはサムネイル表示
- 話者名は太字で上部に表示

### 4. SE（効果音）

テキストの特定位置でSEをトリガーする。

#### 4.1 記法

```markdown
[se src="click.mp3" volume="0.5"]  <!-- インラインSEマーカー -->

:::zw-typing{speed:"50ms", se:"keystroke.mp3"}  <!-- タイピング連動SE -->
テキスト
:::
```

#### 4.2 管理

| 項目 | 仕様 |
|------|------|
| 対応形式 | mp3, ogg, wav |
| 保存先 | ドキュメントに紐づくアセットフォルダ |
| プレビュー | WYSIWYG上でSEマーカーにホバーすると試聴可能 |
| 読者向け | HTML出力に `<audio>` として埋め込み、Intersection Observer でトリガー |
| 音量制御 | 個別音量 + マスター音量 |
| ミュート | 読者側にミュートボタンを提供 |

#### 4.3 WYSIWYG上の表示

- SEマーカーはインラインのスピーカーアイコンとして表示
- クリックで試聴、右クリックで設定変更

### 5. スクロール連動演出

要素がビューポートに入ったときに演出を開始する。

#### 5.1 対応する演出

- フェードイン / スライドイン
- テクスチャアニメーション開始
- タイピング演出開始
- SE再生

#### 5.2 記法

```markdown
[anim type="scroll-trigger" effect="fade-in" delay="200ms"]
スクロールで表示されるテキスト
[/anim]
```

## ジャンルプリセット

複数の演出を組み合わせたプリセットを提供:

| プリセット名 | 含む演出 |
|-------------|----------|
| ADV風 | ダイアログボックス + タイピング演出 + クリック進行 |
| Web小説風 | テクスチャオーバーレイ + スクロール連動 |
| ホラー風 | グリッチ + フェードイン + SE |
| ポエム風 | 文字アニメーション + テクスチャ + 余白制御 |

プリセットは SP-062 の Preset sugar として実装し、
下位レイヤ (TextEffect / Animation / Ornament / Media) に展開する。

---

## 成功状態

- WYSIWYG上でテクスチャオーバーレイが文字に適用され、リアルタイムでアニメーションする
- タイピング演出のプレビューが WYSIWYG 内で再生できる
- ダイアログボックスが WYSIWYG 上でブロック要素として編集できる
- HTML出力で上記すべての演出が読者向けに再生される
- reduced motion 環境で適切に縮退する
