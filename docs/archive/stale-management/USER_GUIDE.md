## はじめに

Zen Writer は、小説や物語の執筆に特化した Web アプリケーションです。シンプルな操作性と洗練されたデザインで、執筆に集中できる環境を提供します。

### 特徴

- **サーバー不要**: ブラウザだけで動作、インターネット接続不要
- **自動保存**: 入力内容は LocalStorage に自動保存
- **モダンダークモード**: 目に優しいダークテーマがデフォルト
- **柔軟なカスタマイズ**: テーマ、フォント、レイアウトを自由に調整
- **強力な検索**: 全ドキュメント横断検索、正規表現対応
- **階層的管理**: ドキュメントをツリー構造で整理

## 基本的な使い方

### 起動方法

#### 開発サーバーで起動（推奨）

```bash
npm run dev
```

ブラウザで `http://localhost:8080` にアクセス

#### 直接ファイルアクセス

`index.html` をブラウザで直接開く（一部機能に制限あり）

### 初回起動時

1. アプリが自動的にダークモードで起動します
2. サイドバーとツールバーが表示されます
3. エディタエリアに文章を入力できます
4. 入力内容は自動的に保存されます

### テキストの入力

- エディタエリア（中央の大きな領域）をクリックして入力開始
- 入力内容は数秒ごとに自動保存されます
- 文字数・語数がツールバーに表示されます

### データの保存

- **自動保存**: 入力内容は LocalStorage に自動保存
- **手動保存**: `Ctrl + S` でも保存可能（実際は自動保存されています）
- **エクスポート**: サイドバー「ドキュメント」から `.txt` / `.md` で保存

## テーマとカスタマイズ

### テーマ切り替え

#### デフォルトテーマ

- 起動時は**モダンダークモード**（VS Code/Notion風）
- 目に優しいダークグレー基調で長時間の執筆にも快適

#### テーマの変更方法

1. **ツールバー右上**のテーマアイコン（月または太陽マーク）をクリック
2. ダークモード ↔ ライトモードが切り替わります
3. スムーズなアニメーション効果で視覚的に洗練

#### キーボード操作

- テーマボタンにフォーカスして `Enter` または `Space` で切り替え

#### テーマの自動保存

- 選択したテーマは LocalStorage に保存
- 次回起動時も同じテーマが適用されます

### カスタムカラー

サイドバーの「テーマ」セクションから：

- **背景色**: カラーピッカーで自由に設定
- **テキスト色**: 読みやすい色を選択
- **リセット**: デフォルト配色に戻す

### フォント設定

1. サイドバー「タイポグラフィ」を開く
2. **フォントファミリー**: 好みのフォントを選択
3. **フォントサイズ**: スライダーまたは数値入力で調整
4. **行間**: 読みやすい行間に設定

#### クイック調整

- `Ctrl/Cmd + +`: フォントサイズ拡大
- `Ctrl/Cmd + -`: フォントサイズ縮小
- `Ctrl/Cmd + 0`: フォントサイズ初期化

## ドキュメント管理

### 階層的ドキュメントツリー

サイドバー「構造」タブで：

- **ドキュメント一覧**: ツリー構造で表示
- **ドラッグ&ドロップ**: ドキュメントを並び替え
- **階層管理**: フォルダのように整理

### 新規ドキュメント作成

1. サイドバー「構造」タブを開く
2. 「新規ドキュメント」ボタンをクリック
3. ドキュメント名を入力
4. 自動的に新しいドキュメントに切り替わります

### ドキュメント切り替え

- ツリー内のドキュメント名をクリック
- 編集中の内容は自動保存されます

### ドキュメントの整理

- ドラッグ&ドロップで順序変更
- 関連するドキュメントをグループ化

## 全文検索

### 検索パネルの起動

- `Ctrl + Shift + F` で全文検索パネルを開く
- または、ツールバーの検索アイコンをクリック

### 検索方法

1. **クエリ入力**: 検索したいキーワードを入力
2. **Enter または検索ボタン**: 検索を実行
3. **結果表示**: マッチしたドキュメントが一覧表示

### 検索オプション

- **大文字小文字**: 区別して検索
- **正規表現**: 高度なパターンマッチング
- **検索履歴**: 最新10件を保存

### 検索結果の利用

- **結果クリック**: ドキュメントを開き、該当箇所をハイライト
- **マッチ数表示**: 各ドキュメントのマッチ数を確認
- **プレビュー**: 検索結果のコンテキストを表示

### 検索パネルを閉じる

- `ESC` キーで閉じる
- パネル外をクリック

## ガジェット機能

### ガジェットとは

サイドバーに配置できる機能モジュールです：

- **時計**: 現在時刻を表示
- **目標設定**: 文字数目標と締切を設定
- **Pomodoroタイマー**: 集中セッション管理
- **文字数カウンタ**: リアルタイム集計
- **アウトライン**: 章構成を管理

### ガジェットの操作

#### 並び替え

- ガジェットヘッダーをドラッグ&ドロップ
- 好みの順序に配置

#### 折りたたみ

- ガジェットヘッダーをクリック
- コンテンツを非表示にしてスペース節約

#### 設定の保存

- ガジェットの配置と状態は自動保存
- 次回起動時も同じレイアウト

### 主要ガジェット

#### 目標設定ガジェット

1. 目標文字数を入力
2. 締切日を設定（オプション）
3. 進捗状況が自動更新

#### Pomodoroタイマー

1. 作業時間を設定（デフォルト: 25分）
2. 休憩時間を設定（デフォルト: 5分）
3. 「開始」ボタンでセッション開始
4. 集中して執筆

## キーボードショートカット

### 基本操作

| ショートカット | 機能 |
|---------------|------|
| `Ctrl + S` / `Cmd + S` | 保存（自動保存されているため確認用） |
| `Tab` | インデント |
| `Alt + W` | ツールバー表示/非表示 |

### フォント調整

| ショートカット | 機能 |
|---------------|------|
| `Ctrl/Cmd + +` | フォントサイズ拡大 |
| `Ctrl/Cmd + -` | フォントサイズ縮小 |
| `Ctrl/Cmd + 0` | フォントサイズ初期化 |

### 検索

| ショートカット | 機能 |
|---------------|------|
| `Ctrl + Shift + F` | 全文検索パネルを開く |
| `ESC` | 検索パネルを閉じる |

### その他

| ショートカット | 機能 |
|---------------|------|
| `F11` またはツールバーアイコン | フルスクリーン切替 |
| テーマボタン + `Enter/Space` | テーマ切り替え |

## 設定とカスタマイズ

### サイドバーの表示/非表示

- ツールバー左端のハンバーガーメニューをクリック
- または画面左端からスワイプ（タッチデバイス）

### ツールバーの表示/非表示

- `Alt + W` で切り替え
- またはツールバー右上のアイコン

### フルスクリーンモード

- ツールバー右端のアイコンをクリック
- `F11` でブラウザのフルスクリーン（推奨）

### 設定の保存と復元

すべての設定は LocalStorage に自動保存されます：

- テーマ設定
- フォント設定
- ガジェット配置
- ドキュメント内容
- ウィンドウレイアウト

### データのバックアップ

#### エクスポート

1. サイドバー「ドキュメント」を開く
2. エクスポートボタンをクリック
3. `.txt` または `.md` 形式で保存

#### インポート

1. サイドバー「ドキュメント」を開く
2. インポートボタンをクリック
3. テキストファイルを選択

### ブラウザを変更した場合

LocalStorage はブラウザごとに管理されます：

- 別のブラウザでは設定が引き継がれません
- エクスポート機能で手動バックアップを推奨

## トラブルシューティング

### データが消えた

- ブラウザのキャッシュクリアで LocalStorage が削除された可能性
- 定期的なエクスポートでバックアップを推奨

### テーマが適用されない

- ブラウザをリロード（`Ctrl + R`）
- LocalStorage をクリアして再起動

### 検索が動作しない

- `Ctrl + Shift + F` でパネルが開くか確認
- ドキュメントに内容があるか確認

### パフォーマンスが遅い

- 大量のドキュメント（100件以上）がある場合
- 不要なドキュメントを削除
- ブラウザのキャッシュをクリア
