## はじめに

Zen Writer は、小説や物語の執筆に特化した Web アプリケーションです。シンプルな操作性と洗練されたデザインで、執筆に集中できる環境を提供します。

### 特徴

- **サーバー不要**: ブラウザだけで動作、インターネット接続不要
- **自動保存**: 入力内容は LocalStorage に自動保存
- **モダンダークモード**: 目に優しいダークテーマがデフォルト
- **柔軟なカスタマイズ**: テーマ、フォント、レイアウトを自由に調整
- **強力な検索**: 全ドキュメント横断検索、正規表現対応
- **階層的管理**: ドキュメントをツリー構造で整理
- **複数のテーマプリセット**: ライト/ダーク/セピア/オブシディアン/フォレスト/オーシャンなど

## 基本的な使い方

### 起動方法

#### 開発サーバーで起動（推奨）

```bash
npm run dev
```

ブラウザで `http://localhost:8080` にアクセス

#### 直接ファイルアクセス

`index.html` をブラウザで直接開く（一部機能に制限あり）

### 初回起動時

1. アプリが自動的にダークモードで起動します
2. サイドバーとツールバーが表示されます
3. エディタエリアに文章を入力できます
4. 入力内容は自動的に保存されます

### テキストの入力

- エディタエリア（中央の大きな領域）をクリックして入力開始
- 入力内容は数秒ごとに自動保存されます
- 文字数・語数がツールバーに表示されます

### データの保存

- **自動保存**: 入力内容は LocalStorage に自動保存
- **手動保存**: `Ctrl + S` でも保存可能（実際は自動保存されています）
- **エクスポート**: サイドバー「ドキュメント」から `.txt` / `.md` で保存

## テーマとカスタマイズ

### テーマ切り替え

#### デフォルトテーマ

- 起動時は**モダンダークモード**（VS Code/Notion風）
- 目に優しいダークグレー基調で長時間の執筆にも快適

#### テーマプリセット

サイドバーの「テーマ」セクションから複数のプリセットを選択できます:

- **ライト**: 明るい背景のテーマ
- **ダーク**: 目に優しいダークグレー
- **セピア**: 落ち着いた温かみのある色調
- **オブシディアン**: 深い黒基調
- **フォレスト**: 緑系統の配色
- **オーシャン**: 青系統の配色

#### テーマプリセットの適用方法

1. ツールバー左端のメニューボタンでサイドバーを開く
2. 「テーマ」セクションを展開（折りたたみになっている場合）
3. プリセットボタンをクリック
4. スムーズなアニメーション効果で視覚的に切り替わります

#### テーマの自動保存

- 選択したテーマは LocalStorage に保存
- 次回起動時も同じテーマが適用されます

### カスタムカラー

サイドバーの「テーマ」セクションから:

- **背景色**: カラーピッカーで自由に設定
- **テキスト色**: 読みやすい色を選択
- **リセット**: デフォルト配色に戻す

プリセット選択時は、背景色/文字色が自動的に反映されます。

### フォント設定

1. サイドバー「タイポグラフィ」を開く
2. **フォントファミリー**: 好みのフォントを選択
3. **フォントサイズ**: スライダーまたは数値入力で調整
4. **行間**: 読みやすい行間に設定

#### クイック調整

- `Ctrl/Cmd + +` または `Ctrl/Cmd + =`: フォントサイズ拡大
- `Ctrl/Cmd + -`: フォントサイズ縮小
- `Ctrl/Cmd + 0`: フォントサイズ初期化

## ドキュメント管理

### 複数ドキュメント管理

- サイドバー「ドキュメント」のドロップダウンから、作成済みのドキュメントを選択できます
- 「作成」で新規ドキュメントを追加、「改名」で現在ドキュメントの名前を変更、「削除」で現在ドキュメントを削除します
- ドキュメントを切り替えると、エディタの内容がそのドキュメントに切り替わります
- 入力した内容は各ドキュメント単位で自動保存され、再読み込み後も保持されます
- ドキュメント一覧は、最終更新日時の新しい順で並びます
- 現在のドキュメント名はブラウザのタブタイトルに反映されます（例: `MyNovel - Zen Writer`）

### 未保存の変更と確認ダイアログ

- 未保存の変更がある状態で「新規ドキュメント」または別ドキュメントへの切り替えを行うと確認ダイアログが表示されます
- 続行を選ぶと現在の内容はバックアップ用スナップショットとして自動退避されます
- バックアップパネルから復元可能です

### 新規ドキュメント作成

1. サイドバー「ドキュメント」を開く
2. 「新規ドキュメント」ボタンをクリック
3. ドキュメント名を入力
4. 自動的に新しいドキュメントに切り替わります（現在のドキュメントは保持されます）

### ドキュメント切り替え

- ドロップダウンからドキュメント名を選択
- 編集中の内容は自動保存されます

### ファイル操作

#### 読み込み

- 「ファイルを読み込む」で `.txt`/`.md` をインポートできます

#### エクスポート

- `.txt` または `.md` でダウンロードできます
- エクスポートファイル名には、現在のドキュメント名が含まれます（例: `MyNovel_20250925_132233.txt`）

#### 印刷

- 「印刷」ボタンで印刷ダイアログを開きます
- 画面UIは非表示になり、本文のみが印刷レイアウトで表示されます

## 全文検索

### 検索パネルの起動

- `Ctrl + Shift + F` で全文検索パネルを開く
- または、ツールバーの検索アイコンをクリック

### 検索方法

1. **クエリ入力**: 検索したいキーワードを入力
2. **Enter または検索ボタン**: 検索を実行
3. **結果表示**: マッチしたドキュメントが一覧表示

### 検索オプション

- **大文字小文字**: 区別して検索
- **正規表現**: 高度なパターンマッチング
- **検索履歴**: 最新10件を保存

### 検索結果の利用

- **結果クリック**: ドキュメントを開き、該当箇所をハイライト
- **マッチ数表示**: 各ドキュメントのマッチ数を確認
- **プレビュー**: 検索結果のコンテキストを表示

### 検索パネルを閉じる

- `ESC` キーで閉じる
- パネル外をクリック

## ガジェット機能

### ガジェットとは

サイドバーに配置できる機能モジュールです:

- **時計**: 現在時刻を表示
- **目標設定**: 文字数目標と締切を設定
- **Pomodoroタイマー**: 集中セッション管理
- **文字数カウンタ**: リアルタイム集計
- **アウトライン**: 章構成を管理
- **ヘルプ**: アプリ内で操作ガイドを閲覧
- **UI Settings**: タブ表示方式やサイドバー幅を調整
- **バックアップ**: スナップショット管理

### ガジェットの操作

#### 並び替え

- ガジェットヘッダーをドラッグ&ドロップ
- 好みの順序に配置

#### 折りたたみ

- ガジェットヘッダーをクリック
- コンテンツを非表示にしてスペース節約

#### 設定の保存

- ガジェットの配置と状態は自動保存
- 次回起動時も同じレイアウト

### 主要ガジェット

#### 目標設定ガジェット

1. 目標文字数を入力
2. 締切日を設定（オプション）
3. 進捗状況が自動更新

#### 執筆目標と進捗バー

- サイドバーの「執筆目標」で、目標文字数と（必要なら）締切日を設定できます
- ツールバー中央付近に目標の進捗バーが表示され、入力文字数に応じて幅が増加します
- 目標を 0（未設定）に戻すと進捗バーは非表示になります
- モバイル幅（768px以下）では進捗バーは自動で非表示になります
- 目標達成時は右下に通知が一度だけ表示されます（HUDも短時間表示）

#### Pomodoroタイマー

1. 作業時間を設定（デフォルト: 25分）
2. 休憩時間を設定（デフォルト: 5分）
3. 「開始」ボタンでセッション開始
4. 集中して執筆

## アウトライン

### 概要

- サイドバー「アウトライン」でプリセットを切替/作成できます
- 部/章/節などの階層構造を管理し、カーソル位置に見出しを挿入できます

### プリセット管理

- 「新しいプリセットを作成」で、カンマ区切りのレベル名（例: `部,章,節`）を登録できます
- 各レベルの色を変更すると、挿入ボタンの表示にも反映されます

### 見出しの挿入

- 挿入ボタンで、カーソル位置にそのレベルの見出しテンプレート（Markdown見出し）を挿入します

### レベルの並べ替え

- レベル行の「↑/↓」ボタンで順序を入れ替えできます（保存されます）

## Visual Profile（見た目のプリセット）

### 概要

- テーマ/背景色/フォント/余白/表示モードといった「見た目」の設定を、「Visual Profile」としてまとめて扱います

### 使い方

- 執筆モードのプリセットを切り替える感覚で一括変更できます
- 詳細な仕様は `docs/VISUAL_PROFILE.md` を参照してください

### 今後の拡張

## ミニHUD（小型表示）

### 概要

- 入力中に左下へ小さなHUDで「文字数/語数」をフェード表示します
- 一時的な情報表示のため、数秒で自動的にフェードアウトします

### HUD 設定

サイドバー「HUD 設定」から次を変更できます:

- **表示位置**: 左下/右下/左上/右上
- **表示時間（ms）**: 300〜5000 の範囲
- **背景色/文字色**: カラーピッカーで自由に設定
- **不透明度**: 0〜1（0.05単位）

### テスト表示

- 「HUDテスト表示」で現在の設定でHUDを一時表示できます（動作確認用）

## フローティングツール（フォント調整）

### 概要

- 右下の「ツール」ボタン（歯車アイコン）で小さなパネルを開閉できます
- スライダーまたは数値入力でページ全体のフォントサイズを変更できます

### 使い方

1. 右下の歯車アイコンをクリック
2. パネルが開きます
3. スライダーを動かす、または数値を入力してフォントサイズを調整

## ヘルプ機能

### 概要

- サイドバーの「アシスト」タブ内に「ヘルプ」ガジェットがあります
- カテゴリボタンをクリックすると、その内容がサイドバー内にシームレスに表示されます
- 外部ページへ遷移せずに操作方法を確認できます

### ヘルプカテゴリ

- **はじめに**: 基本操作とショートカット
- **エディタ機能**: 編集、検索、フォント装飾、タイプライターモード
- **ドキュメント管理**: 複数ドキュメント、ファイル操作、バックアップ、スナップショット設定
- **物語Wiki**: 設定管理、AI自動生成、ツリーペイン、コルクボード
- **ガジェット設定**: ロードアウト、タブ管理、ツリーペイン
- **表示モード**: 通常/フォーカス/ブランク、ツリーペイン
- **テーマ・外観**: テーマ切替、カスタムカラー、背景ビジュアル
- **キーボード操作**: ショートカット一覧、Markdownショートカット
- **Markdown編集**: ライブプレビュー、リスト自動継続、見出し・リンク挿入

## バックアップ（スナップショット）

### 概要

- サイドバー「バックアップ」で本文のスナップショットを保存/復元/削除できます
- 定期的な自動保存と手動保存の両方に対応

### 手動保存

- 「今すぐ保存」ボタンで手動保存

### 自動保存の設定

- 入力中は一定時間（設定可能、初期2分）かつ一定文字数（設定可能、初期300文字）以上の変化があると自動保存されます
- サイドバーの「スナップショット設定」から次を変更できます:
  - 自動保存間隔（分）
  - 差分文字数のしきい値
  - 保持するスナップショット数（初期値: 無制限）

### 復元

- ファイル管理セクションの「最後のスナップショットから復元」ボタンから、直近のスナップショットで本文を復元できます
- この操作では現在の内容も別スナップショットとして退避されるため、復元後も履歴として戻すことができます

### ドキュメント切り替え時のバックアップ

- 未保存の変更がある状態でドキュメントを切り替えると、現在の内容がバックアップ用スナップショットとして自動退避されます

## 設定とカスタマイズ

### 表示モード（UIモード）

ツールバー右側の「表示モード」セレクトボックスから、次のモードを切り替えられます:

- **通常モード**: サイドバー/ツールバーを含む標準レイアウト
- **フォーカスモード**: サイドバーなど一部UIを抑えて、本文への集中を高めます
- **ブランクモード**: ほぼ本文のみのミニマルな画面構成になります（必要に応じて通常モードに戻してください）

### サイドバーの表示/非表示

- ツールバー左端のハンバーガーメニューをクリック
- または画面左端からスワイプ（タッチデバイス）

### ツールバーの表示/非表示

- `Alt + W` で切り替え
- またはツールバー右上のアイコン

### フルスクリーンモード

- ツールバー右端のアイコンをクリック
- `F11` でブラウザのフルスクリーン（推奨）

### タイプライターモード

- サイドバーの設定から、タイプライターモードを有効化できます
- アンカー比（画面のどの位置に入力行を固定するか）と張り付き強度を調整可能

### 設定の保存と復元

すべての設定は LocalStorage に自動保存されます:

- テーマ設定
- フォント設定
- ガジェット配置
- ドキュメント内容
- ウィンドウレイアウト
- UI設定
- スナップショット設定
- 執筆目標
- HUD設定

### 高度なテーマプリセット（テーマバンドル）

- サイドバー「テーマプリセット（拡張）」で、現在の設定（テーマ/フォント/サイズ/行間/カスタム色）を名前を付けて保存できます
- 保存したプリセットはプルダウンから選んで「適用」で反映、「削除」で削除できます

### ブラウザを変更した場合

LocalStorage はブラウザごとに管理されます:

- 別のブラウザでは設定が引き継がれません
- エクスポート機能で手動バックアップを推奨

## トラブルシューティング

### データが消えた

- ブラウザのキャッシュクリアで LocalStorage が削除された可能性
- 定期的なエクスポートでバックアップを推奨
- スナップショット機能を活用してください

### テーマが適用されない

- ブラウザをリロード（`Ctrl + R`）
- LocalStorage をクリアして再起動

### 検索が動作しない

- `Ctrl + Shift + F` でパネルが開くか確認
- ドキュメントに内容があるか確認

### パフォーマンスが遅い

- 大量のドキュメント（100件以上）がある場合
- 不要なドキュメントを削除
- ブラウザのキャッシュをクリア

### 自動保存が効かない

- プライベートブラウズやストレージ制限を確認してください
- ブラウザの LocalStorage が有効か確認

### エクスポート・ダウンロードされない

- ブラウザのポップアップ/ダウンロード設定をご確認ください

### 既知の制約

- ブラウザ/端末間の同期はありません
- 長大なテキストはブラウザのメモリ/LocalStorage制限の影響を受けます
