# Zen Writer エディタ機能ガイド

最終更新: 2026-04-16
更新者: Claude Code

## 概要

Zen Writer のエディタおよび補助機能（HUD、画像挿入、フォント操作など）の利用手順をまとめています。最新の UI と整合しており、新しい端末からアクセスした場合でもここを参照することで操作方法を把握できます。

## 1. 表示モードとVisual Profile

### 1-1. 表示モード（UIモード）

- **通常モード**: サイドバーとヘッダーを表示した標準レイアウト
- **フォーカスモード**: サイドバーを非表示にして執筆に集中
- **ミニマル表示**: 執筆面を優先し、操作は left nav / top chrome / command palette から呼び出す画面

ヘッダー右上のドロップダウンまたはショートカットで切り替えられます。

### 1-2. Visual Profile（表示プロファイル）

- テーマ・フォント・行間・余白バランスをまとめたプリセット機能
- サイドバーの「テーマ・フォント」タブ内の「表示プロファイル」セクションから選択
- プリセット: **ライト** / **ダーク** / **ライト（余白広め）**
- プロファイル切り替え時、表示モードは維持されレイアウトシフトは発生しません

### 1-3. エディタ表示の切り替え（MD プレビュー / リッチ編集 / 再生オーバーレイ）

執筆面や確認面を目的別に使い分けられます。いずれも UI モード（Normal/Focus）とは独立した軸です。

| 名称 | 役割 | 編集可否 |
| ---- | ---- | ------- |
| **MD プレビュー** | 編集画面の横に Markdown のレンダリング結果を並べて確認 | 不可（プレビューは読み取り専用） |
| **リッチ編集（WYSIWYG）** | 装飾を視覚的に編集できる執筆面。既定の執筆面として起動 | 可 |
| **再生オーバーレイ** | 読者視点で原稿の最終的な見え方を確認するための一時オーバーレイ | 不可（「編集に戻る」で執筆面へ復帰） |

**主な切替導線**: left nav、top chrome、command palette（`Ctrl+K` 等）から到達可能です。

**誤解を避けるポイント**:

- 「MD プレビュー」は**編集画面の横並び表示**であり、読者視点の確認ではありません。
- 「再生オーバーレイ」は**読了確認用の一時オーバーレイ**で、UI モードそのものではありません。
- スクリーンリーダー等の支援技術は OS/ブラウザ側の機能であり、再生オーバーレイと無関係です。

## 2. ロードアウト（ガジェット構成）

- サイドバーに表示するガジェットの組み合わせを保存・切り替えできます
- プリセット: **小説・長編** / **ミニマル** / **ビジュアルノベル** / **脚本・シナリオ**
- 「保存」で現在の構成を保存、「複製」で既存ロードアウトをコピー
- 「適用」で選択したロードアウトに切り替え、「削除」で不要なロードアウトを削除

## 3. HUD（ミニ通知）

- ガジェットの「HUD」カードから設定を開きます。
- メッセージ、表示位置、表示時間、背景色/文字色、サイズ、不透明度を調整できます。
- 「ピン留め」を有効にすると常時表示され、無効時はフェードアウトします。
- 「プレビュー」で即時確認、「クリア」でメッセージ削除、「リセット」で初期設定に戻せます。

## 4. 執筆目標

- ガジェット側で表示/非表示や順序を管理できます。
- 目標文字数、締切日を設定すると、ツールバーの進捗バーやHUD通知に連携します。

## 5. タイプライターモード

- カーソル位置を画面中央付近に固定し、スクロールを抑制する機能
- サイドバーの「アシスト」タブ内「Typewriter」ガジェットで設定
- アンカー位置（上/中央/下）と追従度（低/中/高）を調整可能

## 6. 画像の挿入

### 6-1. 貼り付け

1. クリップボードに画像をコピーする。
2. エディタにフォーカスした状態で `Ctrl/⌘ + V`。
3. `![ファイル名](asset://ID)` 形式の Markdown が挿入されます。

### 6-2. ドラッグ＆ドロップ

1. エディタへ画像ファイル（PNG/JPEG 等）をドラッグします。
2. ドロップ時に自動で asset:// 参照の Markdown が挿入されます。
3. 既にテキストがある場合、自動で改行が追加され適切に配置されます。

### 6-3. 画像プレビュー機能

- 挿入された画像はエディタ内で直接表示（オーバーレイ）されます。
- オーバーレイ画像をドラッグして位置を調整できます。
- 右下のリサイズハンドルで幅を変更できます。
- 左上のトグルボタンで表示/非表示を切り替えられます。
- 右側の「画像プレビュー」パネルで一括管理可能です。
  - スライダーで幅調整、プルダウンで整列変更。
  - 「隠す」/「表示する」ボタンで個別制御。
  - 「別タブで開く」でフルサイズ表示。

## 7. フォント・テーマ・カスタムカラー

サイドバーの「テーマ・フォント」タブから調整できます。

### 7-1. テーマとカスタム色

- テーマ: ライト / ダーク / セピア / 高コントラスト / ソラリゼド
- カスタム色: 背景色・文字色を個別に設定可能

### 7-2. フォント切り替え（4軸）

フォント設定は以下の4軸で構成されています。

| 軸 | 操作場所 | 範囲 | CSS変数 |
|----|---------|------|---------|
| 本文フォント | Typography ガジェット | プリセット選択 | `--font-family` |
| 本文サイズ | Typography + Quick Tools | 12-48px | `--editor-font-size` |
| UIサイズ | Typography ガジェット | 12-24px | `--ui-font-size` |
| 行間 | Typography ガジェット | 1.0-3.0 | `--line-height` |

- **Typography ガジェット**: 4軸すべてを詳細に設定できる正規UI
- **Quick Tools（フローティングパネル）**: 本文サイズのみ素早く調整可能。Typography と双方向に同期
- **ショートカット**: `Ctrl/Cmd + Plus/Minus/0` で本文サイズを増減・リセット

設定はリロード後も保持されます。変更時に他の設定が消えることはありません（マージ保存方式）。

### 7-3. 見出しスタイル

サイドバー「テーマ・フォント」セクション内の「見出しスタイル」ガジェットで H1-H6 の見た目を調整できます。

- **プリセット選択**: 「標準」「章扉」「本文重視」の3種類
- **個別調整**: H1-H3 は size/weight/lineHeight/letterSpacing、H4-H6 は size のみ
- **ミニプレビュー**: 調整結果を即座に確認
- **リセット**: カスタム調整を破棄してプリセット値に戻す

見出し設定は印刷時にも自動反映されます。

## 8. Markdown機能

### 8-1. ショートカット

- `Ctrl/⌘ + B`: **太字**
- `Ctrl/⌘ + I`: *斜体*
- `Ctrl/⌘ + K`: リンク挿入

### 8-2. リスト自動継続

- `-` や `1.` で始まる行で Enter を押すと、次の行にも自動でリスト記号が追加されます

### 8-3. ライブプレビュー

- 見出し、強調、リスト、リンクなどがリアルタイムでプレビュー表示されます

## 10. バックアップとスナップショット

- 「今すぐ保存」ボタンでスナップショットを作成
- 自動バックアップ: 間隔（デフォルト2分）と変更量（デフォルト300文字）で自動作成
- 保持数を設定して古いスナップショットを自動削除
- 「復元」ボタンで最新のスナップショットから復元可能

## 11. ドキュメント管理

- 「ドキュメント」ガジェットから複数の原稿を管理
- 「作成」「改名」「削除」ボタンで基本操作
- 「読込」でローカルファイル（.txt/.md）を読み込み
- 「TXT保存」「MD保存」でエクスポート
- 「印刷」「PDF」で印刷プレビュー/PDF出力

## 12. 物語Wiki

- 作品内の設定やキャラクター情報を管理するWiki機能
- サイドバーの「編集」タブから「Story Wiki」にアクセス
- カテゴリ分類（人物/場所/アイテム/組織/イベント/設定/その他+カスタム）
- ツリー+詳細ペインの2ペイン構成
- 用語自動検出とエディタ連携機能

## 14. 章管理とシーンナビゲーション

### 14-1. Focus モードの章チップ

Focus モードでは、サイドバー内に章を横並びで示すチップが表示されます。並び順は**左端のチャプターパネルと同一**で、メモリ上の章一覧から生成されます。

### 14-2. シーン見出し（### 見出し）

章チップの下に表示される「シーン」項目は、**現在の章の本文に含まれる `### 見出し` のみ**を抽出したものです。`##` の章見出しは章チップ側に集約されるため、シーン一覧には含まれません。

### 14-3. 前後シーンナビ

章チップ直下の「前のシーン」「次のシーン」ボタンは、現在カーソル位置より前後にある `### 見出し` へジャンプします。シーンが 1 つもない章では両ボタンとも無効化されます。
