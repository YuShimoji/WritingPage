# 機能別チェックリスト & ステータスレポート

Zen Writer プロジェクトの現在の機能実装状況と、将来のロードマップを網羅したチェックリストです。

## 1. エディタ基本機能

| 機能 | ステータス | 備考 |
| :--- | :--- | :--- |
| テキスト編集 (textarea) | ✅ 実装済み | 基本的な執筆機能 |
| 自動保存 (LocalStorage) | ✅ 実装済み | 入力中の内容をブラウザに保持 |
| 文字数・語数カウント | ✅ 実装済み | サイドバーおよびスタンプ表示 |
| インポート (TXT/MD) | ✅ 実装済み | ローカルファイル読み込み |
| エクスポート (TXT/MD) | ✅ 実装済み | ローカルファイル保存 |
| 印刷レイアウト | ✅ 実装済み | 本文のみのクリアな印刷 |
| 検索・置換 | ✅ 実装済み | エディタ内検索/置換 (Ctrl+F) |
| タイプライターモード | ✅ 実装済み | カーソル位置の高さ固定 |
| Markdown ライブプレビュー | ⚠️ 一部実装 | 見出し、強調、リスト等の最小プレビュー |
| リッチテキスト (WYSIWYG) | ❌ 未実装 | contenteditable ベースの編集 |
| スペルチェック | ❌ 未実装 | ブラウザ標準以外の高度な提案 |

## 2. サイドバー & ガジェット

| 機能 | ステータス | 備考 |
| :--- | :--- | :--- |
| タブ切り替えシステム | ✅ 実装済み | Structure / Typography / Assist / Wiki |
| ガジェット登録基盤 | ✅ 実装済み | モジュール化されたガジェット追加 |
| ロードアウト (構成保存) | ✅ 実装済み | ガジェット配置のプリセット管理 |
| 並び替え・折りたたみ | ✅ 実装済み | ドラッグ＆ドロップおよびボタン操作 |
| 設定インポート/エクスポート | ✅ 実装済み | ガジェット設定を JSON で管理 |
| アウトライン表示 | ✅ 実装済み | 章・シーンの構造化表示 |
| 柔軟なタブ配置 (上下左右) | ⚠️ 一部実装 | Phase E-3/E-4 で完全対応予定 |
| フローティングパネル | ✅ 実装済み | サイドバーからの切り離し・透明度調整 |

## 3. テーマ & 外観

| 機能 | ステータス | 備考 |
| :--- | :--- | :--- |
| カラープリセット | ✅ 実装済み | ライト / ダーク / セピア / ナイト |
| カスタムカラーピッカー | ✅ 実装済み | UI とエディタ個別の配色設定 |
| フォント調整 | ✅ 実装済み | サイズ、行間、余白の調整 |
| Visual Profile | ✅ 実装済み | 外観設定のコレクション保存・適用 |
| 背景グラデーション | ✅ 実装済み | CSS 変数による制御 |
| 背景画像・フィルタ | ❌ 未実装 | スクロール連動背景や詳細フィルタ |

## 4. 特殊機能・支援

| 機能 | ステータス | 備考 |
| :--- | :--- | :--- |
| Story Wiki | ⚠️ 一部実装 | ページ作成・検索。リンク/AIは未実装。 |
| スナップショット (履歴) | ✅ 実装済み | 履歴保存、プレビュー、復元 (Ctrl+Shift+Z) |
| 画像挿入 (Interactive) | ⚠️ 一部実装 | D&D挿入、サイズ/位置調整（基本的） |
| 画像コラージュ | ❌ 未実装 | 複数画像のレイアウト管理 |
| 文章目標 (Goal) | ❌ 未実装 | 執筆目標設定と進捗可視化 |
| Wikilinks ([[link]]) | ⚠️ 一部実装 | 実装済みだが、グラフ表示などは未 |
| コマンドパレット | ❌ 未実装 | 全機能を横断検索・実行 |
