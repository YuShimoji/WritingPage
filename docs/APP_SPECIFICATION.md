## 概要

**製品名：** Zen Writer
**バージョン：** 0.3.29
**種別：** デスクトップアプリ / PWA
**ライセンス：** Private
**開発言語：** JavaScript（バニラJS）

**コンセプト：**
執筆に集中するためのミニマル&スタイリッシュな物語スタジオ。Novlr、Obsidian、Scrivenerの良い点を取り入れた、シンプルでモダンなエディタ。

## 主要機能

### 1. エディタ機能

| 機能 | 説明 | 状態 |
|------|------|------|
| テキストエディタ | シンプルなtextareaベース | 実装済み |
| WYSIWYGエディタ | リッチテキスト編集（切り替え可能） | 実装済み |
| 文字数カウント | リアルタイム表示 | 実装済み |
| 自動保存 | LocalStorageに自動保存 | 実装済み |
| エディタモード | 通常/フォーカス/ブランク | 実装済み |

**エディタモード詳細：**
- **通常モード：** すべてのUIを表示（初心者向け）
- **フォーカスモード：** サイドバー、HUD、FABを非表示
- **ブランクモード：** ツールバーも含めすべて非表示（上級者向け、ショートカット操作）

### 2. ドキュメント管理

| 機能 | 説明 | 状態 |
|------|------|------|
| 階層的ドキュメントツリー | Scrivener/Obsidian風 | 実装済み |
| 全文検索 | 複数ドキュメント横断、正規表現対応 | 実装済み |
| ドラッグ&ドロップ並び替え | ツリー内の順序変更 | 実装済み |
| Wikiリンク | ドキュメント間リンク | 実装済み |

### 3. ガジェットシステム

モジュラー設計による拡張可能なパネルシステム。

| カテゴリ | ガジェット | 説明 |
|---------|-----------|------|
| **構造** | Outline | アウトライン管理 |
|  | Documents | ドキュメント階層 |
|  | Snapshot | バックアップ管理 |
|  | Tags/SmartFolders | タグ・フォルダ管理 |
| **編集** | StoryWiki | 物語Wiki |
|  | Images | 画像管理 |
|  | ChoiceTools | 選択肢ツール |
| **テーマ** | Themes | テーマ切り替え |
|  | Typography | フォント設定 |
|  | VisualProfile | ビジュアルプロファイル |
| **補助** | WritingGoal | 執筆目標 |
|  | HUD | ヘッドアップディスプレイ |
|  | Clock | 時計 |
|  | Pomodoro | 集中タイマー |

**ガジェット総数：** 29個

### 4. 装飾・アニメーション

ビジュアルノベル作成のための視覚的要素。

**装飾機能：**
- 太字、斜体、下線、取り消し線
- 影、輪郭、光彩
- 大文字/小文字変換
- 文字間隔調整

**アニメーション機能：**
- フェード、スライド、タイプライター
- パルス、シェイク、バウンス
- 遅延フェード
- 速度・持続時間調整

### 5. インポート/エクスポート

| 形式 | インポート | エクスポート |
|------|-----------|-------------|
| テキスト（.txt） | ○ | ○ |
| Markdown（.md） | ○ | ○ |
| HTML | × | ○ |
| PDF | × | ○（印刷） |

## UI構成

### ツールバー（11個のボタン）

| グループ | ボタン |
|---------|--------|
| サイドバー制御 | メニュー |
| 情報表示 | 文字数カウント |
| モード | 通常/フォーカス/ブランク切り替え |
| 装飾 | フォント装飾、テキストアニメーション |
| 編集 | プレビュー、分割ビュー、WYSIWYG切り替え |
| システム | 設定、ヘルプ、テーマ切り替え、全画面表示 |

### サイドバー

**タブ構成：** 4タブ（structure/wiki/assist/typography）
**注：** v0.3.29で1タブに統合予定

### メインハブパネル（フローティングパネル統合）

6つの機能をタブ形式で統合：
1. クイックツール（フォントサイズ、HUD制御）
2. 検索/置換
3. 装飾
4. アニメーション
5. 全文検索
6. 分割ビューモード選択

## 配布形式

### 1. Electronアプリ

**App ID：** com.zenwriter.app
**製品名：** Zen Writer
**ビルド方式：** electron-builder

**配布ファイル：**
- `build/win-unpacked/` - Windows実行ファイル
- `Zen Writer.exe` - メインアプリケーション

**インストール：**
- スタートメニューショートカット自動作成
- レジストリ登録なし（ポータブル）

### 2. PWA

**Manifest：**
```json
{
  "name": "Zen Writer",
  "short_name": "ZenWriter",
  "display": "standalone",
  "background_color": "#1e1e1e",
  "theme_color": "#252526"
}
```

**Service Worker：**
- キャッシュ戦略：Network First
- オフライン対応

## 制限事項

### 現在の制限

1. **プラットフォーム：** Windowsのみ正式サポート（Mac/Linux未検証）
2. **同期機能：** なし（LocalStorageのみ）
3. **クラウド保存：** なし
4. **モバイル対応：** デスクトップ優先、モバイルは部分対応
5. **プラグインシステム：** 未実装（v0.4で予定）

### LocalStorage容量

- **標準：** 5-10MB
- **推奨：** 大量のテキストはエクスポートして外部保存
