# Context Toolbar / Shell Controls

> **Status: superseded / history stub**

旧コンテキストツールバー仕様は、現行の統合シェル UI に吸収済み。

現行の操作入口は次を正とする。

- `top chrome`: `F2` / Electron menu / command palette で明示表示
- `left nav`: root/category 階層で category と gadget loadout を選択
- command palette: 横断操作の補助入口
- replay surface: 読者視点確認

新規判断では `docs/INTERACTION_NOTES.md` と `docs/UI_SURFACE_AND_CONTROLS.md` を参照する。
