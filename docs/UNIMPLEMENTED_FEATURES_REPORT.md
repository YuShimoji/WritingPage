## バックログへの追加推奨項目

以下の項目は `docs/BACKLOG.md` に記載がない、または不完全なため追加を推奨:

### 優先度: 高（小説執筆の核心機能）

- [ ] **リッチテキストエディタ（WYSIWYG）** - contenteditableベースのエディタ実装、Markdownとの双方向変換
- [ ] **画像位置調整・サイズ変更** - ドラッグ操作による画像の位置・サイズ変更機能
- [ ] **コラージュレイアウト** - 複数画像の配置・レイアウト機能

### 優先度: 中（執筆体験向上）

- [ ] **テキストアニメーション** - タイピング、フェードイン/アウト等のアニメーション効果
- [ ] **フォント装飾システム** - [bold], [italic], [underline] 等のMarkdown拡張構文とプレビュー
- [ ] **コマンドパレット** - ショートカットと合わせて操作の可視化（検索/置換/ガジェット操作を横断）
- [ ] **分割ビュー** - 編集/プレビュー、章間比較、スナップショット差分表示
- [ ] **フォーカスモード** - 現在行以外を減光/ぼかし（集中支援、タイプライターモードと併用）

### 優先度: 低（長期課題・拡張機能）

- [ ] **Wikilinks/バックリンク/グラフ** - `[[link]]` 構文や `doc://` の可視化・相互参照グラフ
- [ ] **タグ/スマートフォルダ** - ツリーペインにタグ軸、保存された検索、仮想フォルダ
- [ ] **Pomodoro/集中タイマー** - HUD連携のセッション管理
- [ ] **キーバインド編集** - ショートカットの再割当機能
