# 修正が必要な問題

## 1. 文字数カウントが更新されない
- 原因: JavaScriptのイベントリスナーが動作していない可能性
- 対処: ブラウザのコンソールでエラーを確認してください
- 一時的な対処: ページをリロードしてみてください

## 2. テーマカテゴリが開かない/要素がない
- 原因調査中: ガジェット初期化の問題
- 対処: ブラウザのコンソールで以下を実行してください：
  ```javascript
  window.ZWGadgets.init('#theme-gadgets-panel', { group: 'theme' });
  ```

## 3. <>ボタンで消える
- 原因: WYSIWYGモードの切り替えボタン
- 対処: 同じボタンを再度クリックすると元に戻ります
- または: Ctrl+Shift+Wで切り替え可能（既存のショートカット）

## 次のステップ
ブラウザのコンソール（F12）を開いて、エラーメッセージを確認してください。
