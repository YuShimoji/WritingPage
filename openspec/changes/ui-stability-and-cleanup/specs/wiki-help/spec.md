# Wiki Help Capability

## ADDED Requirements

### Requirement: Wiki help access
The system MUST provide a help entry point in the Wiki tab that explains Wiki functionality.

#### Scenario: Help button in Wiki tab
- **GIVEN** Wiki tab is active
- **WHEN** user clicks help button
- **THEN** help content displays in modal or overlay
- **AND** content explains Wiki functionality

#### Scenario: Help content
- **GIVEN** help is open
- **THEN** displays usage instructions for story wiki features
- **AND** includes close button
