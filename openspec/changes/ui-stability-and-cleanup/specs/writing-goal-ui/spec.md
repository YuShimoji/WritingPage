# Writing Goal UI Capability

## ADDED Requirements

### Requirement: Writing goal UI feedback and customization
The system MUST avoid sudden default styling changes (e.g., blue bars) and MUST allow users to control goal UI appearance.

#### Scenario: Input feedback
- **WHEN** user enters number in writing goal
- **THEN** no sudden blue bar appears
- **AND** changes are smooth

#### Scenario: Customization
- **WHEN** user configures writing goal settings
- **THEN** colors and emojis can be controlled
- **AND** default emojis are removed
